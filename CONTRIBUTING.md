# Contributing to MiuuJS

Thank you for your interest in contributing to MiuuJS! We welcome contributions from the community.

## How to Contribute

1. **Fork the repository**
2. **Create a feature branch** (`git checkout -b feature/amazing-feature`)
3. **Make your changes**
4. **Test your changes** by building the frontend assets
5. **Commit your changes** (`git commit -m 'Add some amazing feature'`)
6. **Push to the branch** (`git push origin feature/amazing-feature`)
7. **Open a Pull Request**

## Development Setup

1. Clone the repository to your Pterodactyl panel installation
2. Run `npm install` to install dependencies
3. Run `npm run build` to build frontend assets

## Code Style

- Follow existing code patterns in the project
- Use meaningful commit messages
- Keep pull requests focused on a single change

## Reporting Issues

Report issues using the GitHub issue tracker. Include as much detail as possible:
- Pterodactyl version
- Browser version
- Steps to reproduce
- Expected vs actual behavior

## Pull Request Process

1. Update the README.md with details of changes if needed
2. Ensure the frontend builds without errors
3. Your PR will be reviewed by maintainers

Thank you for contributing!
